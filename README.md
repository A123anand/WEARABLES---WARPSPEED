 # WEARABLES---WARPSPEED

This is the repository for all code used for the Colorado Space Grant Consortium (COSGC) program WARPSPEED team Wearables. 
Team Name: Y.O.D.A (Your Orthopedic Dynamic Alignment)

Electronics:
2 x MPU-6050 6 degrees of freedom cheap IMU 
1 x ESP 32 microcontroller
1 x Battery
1 x Buzzer 
1 x SD card reader

BUZZWORDSMAXXING:

The Madgwick filter is a computationally efficient, quaternion-based sensor fusion algorithm that combines data from gyroscopes,
accelerometers, and optionally magnetometers to estimate device orientation
